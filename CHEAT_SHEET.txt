QUICK START
-----------
# Install dependency:
pip install pillow

# Basic usage:
screensnap                    # Auto-name with timestamp
screensnap myfile.png         # Named screenshot
python screensnap.py          # If not installed globally

COMMON COMMANDS
---------------
screensnap                              # Full screen, auto-name
screensnap screenshot.png                # Full screen, named
screensnap --window "Chrome"             # Capture Chrome window (Windows)
screensnap --output-dir ~/screenshots    # Save to specific directory
screensnap --format jpg myfile.jpg       # Save as JPEG
screensnap --help                        # Show help
screensnap --version                     # Show version

PYTHON API
----------
from screensnap import ScreenSnap

# Create instance:
snap = ScreenSnap()

# Basic capture:
filepath = snap.capture()                    # Auto-name
filepath = snap.capture("myfile.png")        # Named

# Window capture (Windows only):
filepath = snap.capture_window("Chrome")     # Find window by title

# Configure:
snap = ScreenSnap(output_dir="~/screenshots", format="jpg")

TOP 5 USE CASES
---------------
1. UI Troubleshooting
   When: User reports UI bug, need visual feedback
   Command: screensnap bug_report.png
   Time Saved: 10-20 minutes per issue

2. Step-by-Step Instructions
   When: Explaining UI steps to user
   Command: screensnap step1.png (repeat for each step)
   Time Saved: 5-10 minutes per explanation

3. Error Documentation
   When: Capturing error screens for logs
   Command: screensnap error_$(date +%Y%m%d_%H%M%S).png
   Time Saved: 2-5 minutes per error

4. System State Capture
   When: Documenting system configuration
   Command: screensnap --window "Settings" config.png
   Time Saved: 5 minutes per capture

5. Automated Error Capture
   When: Script encounters error
   Python: snap.capture(f"error_{timestamp}.png")
   Time Saved: Instant visual debugging

CONFIGURATION
-------------
Config file: ~/.screensnaprc
Format: JSON

{
  "output_dir": ".",
  "format": "png",
  "include_timestamp": true
}

TROUBLESHOOTING
---------------
Error: "Pillow is required"
Fix: pip install pillow

Error: "Failed to capture screenshot"
Fix: Check you have a display (not SSH without X11)

Error: "Invalid filename"
Fix: Use only alphanumeric, underscore, hyphen in filenames

Error: "Permission denied"
Fix: Check write permissions on output directory

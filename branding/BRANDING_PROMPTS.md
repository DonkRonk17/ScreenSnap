# ScreenSnap Branding Prompts

## Design System: Beacon HQ Visual System v1

**Core Principles:**
- Clean, modern, professional
- Consistent geometric patterns (dodecahedrons, sacred geometry)
- Color palette: Deep blues (#1a237e), teals (#00bcd4), golds (#ffd700)
- Minimalist but distinctive

**Tool Concept:** Screenshot capture tool for fast visual feedback during troubleshooting

**Visual Metaphors:**
- Camera shutter / viewfinder: Represents screenshot capture
- Frame / window border: Represents screen/window capture
- Lightning bolt / instant: Represents speed and immediacy
- Crosshair / target: Represents precision

**Color Psychology:**
- Teal (#00bcd4): Technology, precision, clarity, visual focus
- Gold (#ffd700): Value, important moments, highlight
- Deep blue (#1a237e): Professional, stable, trustworthy
- Dark background (#0a0a1a): Tech-focused, non-intrusive

**Target Aesthetic:** Sharp, precise, instant - like a professional camera interface meets modern dev tools

**Aspect Ratio:** 16:9 (1920x1080 recommended)  
**Purpose:** README header, documentation, presentations  
**File Name:** `titlecard_screensnap.png`

**DALL-E Prompt:**
```
Create a professional title card for a software tool called "ScreenSnap".

Visual style: Clean, modern, tech-focused with sharp geometric elements. Use a dark background (#0a0a1a) with vibrant accent colors (teal #00bcd4, gold #ffd700).

Main elements:
- Bold, clean title "ScreenSnap" in modern sans-serif font (left-center)
- Tagline: "Instant Screenshot Capture" in smaller text below title
- Right side: Abstract representation of a camera shutter or viewfinder frame with geometric patterns
- Subtle crosshair or targeting reticle integrated into the design
- Sharp, angular lines suggesting precision and speed
- Minimal dodecahedral or sacred geometry elements in background
- Lightning bolt or speed lines suggesting instant capture

Layout: Horizontal 16:9 format, title on left third, geometric elements on right two-thirds.

Style: Professional, minimal, sharp and precise. Think "professional camera UI meets modern dev tools". Emphasis on speed and accuracy.
```

## Design Notes

**Brand Identity:**
ScreenSnap is about precision, speed, and clarity. The visual identity should convey:
- **Instant** - Fast capture, no waiting
- **Precise** - Captures exactly what you need
- **Professional** - Trustworthy tool for serious work
- **Technical** - Built for developers and tech professionals

**Visual Language:**
- Sharp angles (not rounded) - precision
- Crosshairs/reticles - targeting/accuracy
- Camera shutter elements - capture/snapshot
- Speed lines/lightning - instant action
- Clean typography - professional and modern

**Color Meaning:**
- **Teal**: Primary brand color - represents visual clarity, technology, precision
- **Gold**: Accent color - represents valuable moments, important captures
- **Deep Blue**: Supporting color - professional, trustworthy, stable
- **Dark Background**: Non-intrusive, tech-focused, puts emphasis on content

**Differentiation:**
Unlike generic screenshot tools, ScreenSnap emphasizes:
- Speed (instant capture)
- Precision (targeting what matters)
- Professional use (troubleshooting, documentation)
